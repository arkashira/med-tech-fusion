# Engineer Platform

A simple platform for finding and engaging with certified engineers.

## Usage

1. Create an instance of the `EngineerPlatform` class.
2. Add engineers to the platform using the `add_engineer` method.
3. Search for engineers using the `search_engineers` method.
4. Get an engineer's profile using the `get_engineer_profile` method.
5. Send a message to an engineer using the `send_message` method.

## Testing

Run the tests using `pytest` to ensure the platform is working correctly.
