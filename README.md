# Med Tech Fusion Project
This project demonstrates the core integration feature of medical-grade software with hardware, firmware, and cloud components.

## Requirements
- Python 3.9+
- pytest 6.2+

## Usage
1. Clone the repository
2. Install dependencies using `poetry install`
3. Run the project using `python src/med_tech_fusion.py`
4. Run tests using `pytest tests/test_med_tech_fusion.py`
